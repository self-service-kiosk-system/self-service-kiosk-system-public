# Frontend

Este proyecto es la interfaz del sistema. Está construido con **Vite + React**.

---

## Requisitos previos

Antes de comenzar, asegúrate de tener instalados:

- [Node.js] (v20.19 o superior recomendado)  
- [npm] (v9 o superior)

Verificar con:

```bash
node -v
npm -v
```
Desde /front-app

instalar dependencias:

```bash
npm install
```

correr servidor:

```bash
npm run dev
```

ir a la url mostrada en terminal.

Para correr test:
```bash
npx vitest
```
